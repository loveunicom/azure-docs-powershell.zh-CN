## <a name="context-persistence-and-psjobs"></a><span data-ttu-id="74807-108">上下文持久性和 PSJob</span><span class="sxs-lookup"><span data-stu-id="74807-108">Context Persistence and PSJobs</span></span>

<span data-ttu-id="74807-109">由于 PSJob 作为单独的进程运行，因此必须与 PSJob 共享 Azure 连接。</span><span class="sxs-lookup"><span data-stu-id="74807-109">Since PSJobs are run as separate processes, your Azure connection must be shared with them.</span></span> <span data-ttu-id="74807-110">使用 `Connect-AzureRmAccount` 登录到 Azure 帐户后，将上下文传递给某个作业。</span><span class="sxs-lookup"><span data-stu-id="74807-110">After signing in to your Azure account with `Connect-AzureRmAccount`, pass the context to a job.</span></span>

```azurepowershell-interactive
$creds = Get-Credential
$job = Start-Job { param($context,$vmadmin) New-AzureRmVM -Name MyVm -AzureRmContext $context -Credential $vmadmin} -Arguments (Get-AzureRmContext),$creds
```

<span data-ttu-id="74807-111">但是，如果你已选择使用 `Enable-AzureRmContextAutosave` 自动保存上下文，则上下文会自动与所要创建的任何作业共享。</span><span class="sxs-lookup"><span data-stu-id="74807-111">However, if you have chosen to have your context automatically saved with `Enable-AzureRmContextAutosave`, the context is automatically shared with any jobs you create.</span></span>

```azurepowershell-interactive
Enable-AzureRmContextAutosave
$creds = Get-Credential
$job = Start-Job { param($vmadmin) New-AzureRmVM -Name MyVm -Credential $vmadmin} -Arguments $creds
```

## <a name="automatic-jobs-with--asjob"></a><span data-ttu-id="74807-112">使用 `-AsJob` 的自动作业</span><span class="sxs-lookup"><span data-stu-id="74807-112">Automatic Jobs with `-AsJob`</span></span>

## <a name="example-scenarios"></a><span data-ttu-id="74807-119">示例方案</span><span class="sxs-lookup"><span data-stu-id="74807-119">Example Scenarios</span></span>

<span data-ttu-id="74807-120">一次性创建多个 VM：</span><span class="sxs-lookup"><span data-stu-id="74807-120">Create several VMs at once:</span></span>

```azurepowershell-interactive
$creds = Get-Credential
# Create 10 jobs.
for($k = 0; $k -lt 10; $k++) {
    New-AzureRmVm -Name MyVm$k  -Credential $creds -AsJob
}

# Get all jobs and wait on them.
Get-Job | Wait-Job
"All jobs completed"
Get-AzureRmVM
```
