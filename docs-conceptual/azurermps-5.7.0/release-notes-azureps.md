# <a name="azure-powershell-570"></a>Azure PowerShell 5.7.0

Azure PowerShell 5.7.0 安装程序：[链接](https://github.com/Azure/azure-powershell/releases/download/v5.7.0-April2018/azure-powershell.5.7.0.msi)

ARM Cmdlet 的库模块：[链接](https://www.powershellgallery.com/packages/AzureRM/5.7.0)

若要从 PowerShell 库安装 `AzureRM`，请运行以下命令：

```powershell-interactive
Install-Module -Name AzureRM -Repository PSGallery -Force
```

若要从旧版 `AzureRM` 更新，请运行以下命令：

```powershell-interactive
Update-Module -Name AzureRM
```
